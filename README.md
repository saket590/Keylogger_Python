# Keylogger_Python

A simple keylogger which can capture the keystrokes.

Usage:

1. install pynput with 'pip3 install pynput'
2. Run the main file script.py 
3. The code will capture the keystrokes and will record them in the keylogger text file in the Desktop.
4. In order to change the file location, kindly change the path in the code.
